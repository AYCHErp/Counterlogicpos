CounterLogic Point of Sale for webERP
November 2014

By Logic Works Limited
http://www.logicworks.co.nz

CounterLogic POS is developed as a POS front end to webERP - backend functionality is all within webERP and CounterLogic deals only with the customer facing sales and returns transactions in conjunction with a barcode scanner, receipt printer and cash drawer.

NOTE: This is NOT free software and a small annual license is required to use CounterLogic POS. 
Please contact support@logicworks.co.nz if you require a license key, please attach a screenshot of your options dialog for each POS terminal that you require a registration key for.
